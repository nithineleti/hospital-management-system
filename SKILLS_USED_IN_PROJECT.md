# Skills Used In This Project

This file summarizes the technical skills and engineering practices used in this codebase.

## Core Backend Skills
- Java (object-oriented programming)
- Spring Boot application development
- Spring MVC (controllers and web flow)
- Spring Data JPA (data access layer)
- Hibernate ORM (entity mapping and persistence)

## Security Skills
- Spring Security configuration
- Form-based authentication
- Role-based access control (RBAC)
- Password hashing with BCrypt
- Secure endpoint mapping

## Database Skills
- Relational database design concepts
- H2 in-memory database setup for development
- Entity relationships and repository patterns
- Query modeling through JPA repositories

## API Skills
- REST API design (multiple `*ApiController` classes)
- JSON serialization/deserialization with Jackson
- Request validation with Spring Validation

## Frontend Skills (Server-Rendered)
- Thymeleaf template engine
- HTML/CSS/JavaScript integration in Spring Boot
- UI organization for feature modules (patient, doctor, dashboard, etc.)

## Build and Dependency Management Skills
- Maven project structure and lifecycle
- Dependency and plugin management in `pom.xml`
- Spring Boot Maven plugin usage

## Quality and Maintenance Skills
- Layered architecture (controller/service/repository/model)
- Configuration management via `application.properties`
- Logging and debugging practices
- Documentation and project cleanup practices

## Security and Maintenance Work Completed Recently
- Dependency vulnerability remediation work done for high-severity CVEs.
- Project dependency baseline updated (Spring Boot and MySQL Connector path).
- Build/run flow validated in the local environment.
